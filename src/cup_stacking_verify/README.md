# 컵 점유 검증 노드

## 개요
컵 점유 검증 노드는 디지털 트윈 환경에서 가상 컵의 점유 상태를 검증하기 위해 설계된 ROS2 기반 노드입니다. 이 노드는 검출 데이터를 처리하고, 검출된 컵과 가상 컵 간의 겹침 비율(overlap ratio)을 계산하며, 결과를 RViz에서 시각화합니다. 이 노드는 Python으로 구현되었으며, ROS2 토픽을 통해 통신합니다.

---

## 입력 파이프라인

### 입력 소스
- **검출 데이터**: `/detected_cups` 토픽을 구독하여 `Detection3DArray` 메시지 형식의 검출 데이터를 수신합니다. 이 메시지는 일반적으로 YOLO 기반 객체 검출 모델 또는 유사한 검출 시스템의 출력입니다.

### 구독하는 토픽
- **`/detected_cups`**:
  - **메시지 타입**: `vision_msgs/msg/Detection3DArray`
  - **설명**: 각 객체의 바운딩 박스 정보(위치 및 크기)를 포함한 검출 객체 배열.
  - **목적**: 가상 컵의 점유 상태를 검증하기 위한 입력 데이터 제공.

---

## 처리 파이프라인

### 주요 함수
1. **`detection_callback`**:
   - 수신된 검출 데이터를 처리합니다.
   - 검출된 컵과 가상 컵 간의 겹침 비율을 계산합니다.
   - RViz에서 시각화를 위해 마커를 퍼블리시합니다.

2. **`calculate_overlap_ratio`**:
   - 검출된 컵과 가상 컵 간의 IoU(Intersection-over-Union)를 계산합니다.
   - 겹침 비율을 실수(float) 값으로 반환합니다.

3. **`get_virtual_box`**:
   - 가상 컵의 예상 위치를 기반으로 바운딩 박스 좌표를 생성합니다.

4. **마커 생성**:
   - **검출된 컵**: 파란색 마커로 표시.
   - **가상 컵**: 초록색 마커로 표시.
   - **텍스트 마커**: 각 컵 위에 겹침 비율을 표시.

---

## 출력 파이프라인

### 퍼블리시하는 토픽
- **`/virtual_cup_markers`**:
  - **메시지 타입**: `visualization_msgs/msg/MarkerArray`
  - **설명**: 검출된 컵과 가상 컵, 겹침 비율 텍스트 마커를 퍼블리시.
  - **목적**: RViz에서 검출 및 검증 과정을 시각화.

- **`/cup_occupancy_status`**:
  - **메시지 타입**: `std_msgs/msg/Int8`
  - **설명**: 각 컵의 점유 상태를 퍼블리시(1: 점유, 0: 비어 있음).
  - **목적**: 각 컵의 점유 상태를 바이너리 값으로 제공.

- **`/cup_overlap_ratio`**:
  - **메시지 타입**: `std_msgs/msg/Float32`
  - **설명**: 모든 검출된 컵 중 최대 겹침 비율을 퍼블리시.
  - **목적**: 의사결정을 위한 최대 겹침 비율 모니터링.

---

## RViz 시각화
- **마커**:
  - **파란색**: 검출된 컵을 나타냄.
  - **초록색**: 가상 컵을 나타냄.
  - **흰색 텍스트**: 각 컵 위에 겹침 비율 표시.
- **프레임**: 모든 마커는 `base_link` 프레임에서 퍼블리시.

---

## 파라미터
- **`p_start`**:
  - **타입**: List[float]
  - **설명**: 가상 컵의 시작 위치.
- **`v_dir`**:
  - **타입**: List[float]
  - **설명**: 가상 컵을 정렬하기 위한 방향 벡터.
- **`threshold`**:
  - **타입**: Float
  - **설명**: 점유 상태를 결정하기 위한 겹침 비율 임계값.
- **`target_index`**:
  - **타입**: Int
  - **설명**: 타겟 가상 컵의 인덱스.

---

## 노드 워크플로우
1. `/detected_cups`를 구독하여 검출 데이터를 수신.
2. 각 검출된 컵을 처리하여 가상 컵과의 겹침 비율을 계산.
3. `/virtual_cup_markers`에 마커를 퍼블리시하여 시각화.
4. 각 컵의 점유 상태와 겹침 비율을 해당 토픽에 퍼블리시.

---

## 향후 개선 사항
- YOLO 또는 기타 객체 검출 모델과의 통합을 통한 실시간 검출 지원.
- 추가 마커 타입 또는 애니메이션을 활용한 시각화 개선.
- 동적 가상 컵 구성 지원.

## 의존성
- **ROS2**: 통신을 위한 미들웨어.
- **RViz**: 마커 시각화 도구.
- **Python**: 노드 구현을 위한 프로그래밍 언어.

## 실행 방법
1. 워크스페이스 빌드:
   ```bash
   colcon build
   ```
2. 워크스페이스 소스:
   ```bash
   source install/setup.bash
   ```
3. 노드 실행:
   ```bash
   ros2 run cup_stacking_verify verifier_node
   ```
4. RViz에서 `/virtual_cup_markers`를 구독하여 시각화.
